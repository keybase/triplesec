~ (void) bulk_encrypt({block_cipher, iv, input, progress_hook, what}, cb)

Given a block cipher and an IV, encrypt the given input in CTR-mode. Do it an an async-fashion so as not to lock up the process.

Parameters:

  • block_cipher ( BlockCipher ) — an instantiated, initialized block cipher
  • iv ( WordArray ) — The random initial value.
  • input ( WordArray ) — The to encrypt
  • progress_hook ( Function ) — A standard progress hook.
  • what ( String ) — Report to the progress_hook what's going on (e.g. "AES" or "Twofish")
  • cb ( callback ) — Callback completed with a WordArray that's the ciphertext
