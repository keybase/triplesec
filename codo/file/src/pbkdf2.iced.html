Method Details

~ (void) pbkdf2({key, salt, klass, c, dkLen, progress_hook}, cb)

A convenience method to make a new PBKDF2 object, and then run it just once.

Parameters:

  • key ( WordArray ) — The secret key/passphrase
  • salt ( WordArray ) — Salt to add to the input to prevent rainbow-table attacks
  • c ( number ) — The number of iterations to run for
  • dkLen ( Number ) — The total length of the derived key material (in bytes)
  • klass ( Class ) — The class to use for inner HMAC, called once per iter. Default is HMAC-SHA512
  • cb ( callback ) — Calls back with a WordArray of key-material.
