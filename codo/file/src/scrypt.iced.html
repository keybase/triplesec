~ (void) scrypt({key, salt, r, N, p, c0, c1, c, klass, progress_hook, dkLen}, cb)

A convenience method to make a new Scrypt object, and then run it just once.

Parameters:

  • key ( WordArray ) — The secret/passphrase
  • salt ( WordArray ) — Salt to add to the intput to prevent rainbow-tables
  • r ( number ) — The r (memory size) parameter for scrypt [default: 8]
  • N ( number ) — The N (computational factor) parameter for scrypt [default : 2^10]
  • p ( number ) — The p (parallellism) factor for scrypt [default : 1]
  • c ( number ) — The number of times to run PBKDF2 [default: 1]
  • c0 ( number ) — The number of times to run PBKDF2 in the 0th pass [ default: 1]
  • c1 ( number ) — The number of times to run PBKDF2 in the 1st pass [ default: 1]
  • klass ( Class ) — The PRF to use as a subroutine in PBKDF2 [default : HMAC-SHA256]
  • progress_hook ( function ) — A Standard progress hook
  • dkLen ( number ) — the length of the derived key.
  • cb ( calllback ) — Calls back with a WordArray of key-material
