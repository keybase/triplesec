~ (WordArray) sign({key, input, hash_class})

A convenience method that makes a new HMAC instance, the signs, and scrubs.

Parameters:

  • key ( WordArray ) — The key to when signing.
  • input ( WordArray ) — The input to sign.
  • hash_class ( Class ) — The class to use for the hasher in the HMAC (SHA512 by default).

Returns:

  • ( WordArray ) — The signature

~ (void) bulk_sign({key, input, progress_hook, klass, what, combine_klasses}, cb)

Sign a lot of data using the async-interface. By default use HMAC-SHA512, but this function is also available to interesting HMAC combinations.

Parameters:

  • key ( WordArray ) — The secret HMAC key
  • input ( WordArray ) — The input to sign.
  • progress_hook ( Function ) — A standard progress hook (optional).
  • klass ( Class ) — The class to allocate for this HMAC, which is HMAC (with SHA512) by default.
  • what ( String ) — What to call this hasher for the sake of the progress hook.
  • cb ( callback ) — Callback with the generated hash, in a WordArray
