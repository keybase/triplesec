    Method Details

    + (void) encrypt({key, data, rng, progress_hook}, cb)

    A convenience wrapper for:

    1. Creating a new Encryptor instance with the given key
    2. Calling run just once.
    3. Scrubbing and deleting all state.

    Parameters:

    • key (Buffer) — The secret key. This data is scrubbed after use, so copy it if you want to keep track of it.
    • data (Buffer) — The data to encrypt. Again, this data is scrubber after use, so copy it if you need it later.
    • rng (Function) — A function that takes as input n and outputs n truly random bytes. You must give a real RNG here and not something fake. You can try require('./prng').generate_words for starters.
    • cb (callback) — Callback with an (err,res) pair. The err is an Error object (if encountered), and res is a Buffer object (on success).
