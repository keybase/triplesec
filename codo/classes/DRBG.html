    Class: DRBG

    Defined in: src/drbg.iced

    Overview

    Implements an HMAC_DRBG (NIST SP 800-90A) based on HMAC_SHA512 Supports security strengths up to 256 bits. Parameters are based on recommendations provided by Appendix D of NIST SP 800-90A. Implementation ported from: https://github.com/fpgaminer/python-hmac-drbg

    Instance Method Summary

    • - (void) constructor(entropy, personalization_string, hmac_func) Constructor Seed the DRBG with sufficient entropy -- 384 bits for initial seed, and 256 for a reseed.
    • - (void) reseed(entropy) Reseed the DRBG with yet more entropy.
    • - (WordArray) generate(num_bytes) Generate bytes from the DRBG.
