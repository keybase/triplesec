    Class: StreamCipher

    Defined in: src/algbase.iced

    Overview

    A base class for a stream cipher. This will be used for bonafide stream ciphers like Salsa20 but also for BlockCiphers running in CTR mode.

    Instance Method Details

    - (Number) encryptBlock(word_array, dst_offset = 0)

    Encrypt one block's worth of data. Use the next block in the keystream (order matters here!). Call into the virtual @get_pad() function to get the pad from the underlying block cipher for this block.

    Parameters:

    • word_array (WordArray) — The WordArray to operator on
    • dst_offset (Number) — The offset to operate on, in wordGs

    Returns:

    • (Number) — the number of blocks encrypted

    - (WordArray) encrypt(word_array)

    Encrypt an entire word array in place, overwriting the original plaintext with the cipher text.

    Parameters:

    • word_array (WordArray) — The plaintext and also the ciphertext location.

    Returns:

    • (WordArray) — Return word_array too just for convenient chaining.

    - (void) bulk_encrypt({input, progress_hook, what}, cb)

    Like encrypt but with an asynchronous preemptable interface. Good for encrypting big payloads without blocking up the process. As above, encrypt in place, so the output ciphertext will be where the input plaintext was.

    Parameters:

    • input (WordArray) — The input cipher text
    • progress_hook (Function) — A standard progress hook
    • what (String) — What the progress hook should say we are doing.
    • cb (callback) — Callback with the completed ciphertext after completion.
